Bienvenido a Phoenix Intranet

Este es un sitio en el que los estudiantes de La Prepa pueden tener un ambiente regulado fuera del colegio para interactuar entre ellos. Este sitio web tiene normas que deben ser acatadas que están presentes en el Acuerdo de licencia del Usuario Final, en el que se encuentran condiciones para usar este sitio en su totalidad.

Este sitio web es un proyecto estudiantil de código abierto y no tiene fines de lucro. El objetivo de este sitio es crear un ambiente seguro para que los estudiantes puedan interactuar entre ellos y compartir sus ideas, opiniones y proyectos. Puede acceder al código fuente presionando Ctrl + U o haciendo clic derecho y seleccionando "Ver código fuente de la página".
